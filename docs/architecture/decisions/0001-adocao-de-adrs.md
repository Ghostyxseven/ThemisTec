# 1. Adoção de ADRs

Data: 2026-06-18

## Status
Aceito

## Contexto
Precisamos de uma forma eficiente para documentar as decisões arquiteturais do projeto Themis, permitindo que todos os membros da equipe (Marcos, Micael, Josiane e Gisele) compreendam por que certas tecnologias ou padrões foram escolhidos durante o desenvolvimento do MVP.

## Decisão
Adotaremos os Architecture Decision Records (ADRs) para registrar as escolhas arquiteturais. Todos os ADRs ficarão armazenados no repositório na pasta `docs/architecture/decisions/`.

## Consequências
- Fica mais fácil integrar novos membros ou justificar decisões para stakeholders.
- Requer disciplina da equipe para sempre que tomar uma decisão crítica, documentar usando este formato.
