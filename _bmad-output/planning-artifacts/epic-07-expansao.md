# Épico 07: Expansão - PWA, Multi-Tenancy e Crawler

## Visão Geral
Este épico foca na modernização da infraestrutura do ThemisTec, adicionando funcionamento offline via PWA, suporte a escritórios com múltiplos usuários (Multi-Tenancy) e infraestrutura base de notificações push via Crawler.

## Histórias
- **Story 37:** Configuração de PWA e Background Sync (IndexedDB).
- **Story 38:** Refatoração de Banco de Dados para Multi-Tenancy e Políticas RLS.
- **Story 39:** Setup de Edge Functions para Crawler de Tribunais e Web Push.
