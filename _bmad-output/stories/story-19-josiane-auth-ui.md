# [STORY] Identidade Visual da Autenticação (US19)

**Issue:** #48
**Responsável:** Josiane (Frontend)
**Épico:** 05 (Revitalização de UI/UX)

## Descrição
Como advogado acessando a plataforma, eu quero me deparar com uma tela de login e cadastro esteticamente premium, moderna e que transmita segurança e sofisticação institucional, para que eu sinta confiança em armazenar dados sensíveis dos meus clientes no ThemisTec.

## Critérios de Aceite
1. O layout das telas de login (`/login`) e cadastro (`/register`) deve abandonar o formato básico de "card centralizado".
2. Implementar um layout "Split-screen" no desktop: metade formulário e metade área de branding (com cor primária, logo e uma frase de efeito jurídica/profissional).
3. O formulário deve apresentar uma tipografia moderna, botões com animações suaves e campos de texto bem espaçados.
4. No mobile, o layout deve se adaptar fluidamente, exibindo apenas o formulário com a logo.
