# História 6.1: Criação do Design System Moderno

**Como** Desenvolvedora Frontend / Designer (Josiane),
**Eu quero** construir um Design System moderno, vibrante e responsivo,
**Para que** a interface do ThemisTec passe uma sensação premium, melhorando a experiência do usuário e destacando-se das ferramentas antigas e defasadas do mercado jurídico.

## Critérios de Aceite:

- **Dado** o ambiente do projeto Next.js com TailwindCSS
- **Quando** a configuração global for carregada
- **Então** ela deve contar com um sistema de cores aprimorado (Tailwind Config), fontes modernas (como Inter, Outfit ou Roboto) e utilitários de glassmorphism e micro-interações.
- **E** devem ser criados os componentes base de UI (Botões, Inputs, Cards) dentro da pasta `src/components/ui/` utilizando essas novas diretrizes de design.
- **E** a pasta `design-artifacts/D-Design-System` deve ser atualizada com os tokens de design (cores, tipografia, espaçamento).
