# Story 30: Acessibilidade e consistência responsiva

**Status:** done

## Story

**Como** usuário do ThemisTec em desktop, celular ou com navegação por teclado,
**quero** páginas consistentes e controles acessíveis,
**para que** eu consiga operar o sistema com clareza em diferentes dispositivos.

## Critérios de aceitação

- [x] O conteúdo principal deve possuir atalho de navegação por teclado.
- [x] Controles interativos devem exibir foco visível consistente.
- [x] O sistema deve respeitar a preferência por movimento reduzido.
- [x] Sidebar, notificações e botões apenas com ícone devem possuir nomes acessíveis.
- [x] Menus expansíveis devem expor estado e relacionamento via ARIA.
- [x] O layout autenticado deve usar regiões semânticas e manter navegação móvel funcional.
- [x] Harness, build e E2E devem passar.
